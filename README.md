Hệ Thống Quản Lý Bán Vé Máy Bay (Aircraft Management System)
📝 Giới thiệu
Dự án được xây dựng để hỗ trợ quản lý quy trình bán vé máy bay, bao gồm các nghiệp vụ từ đặt vé của khách hàng đến quản trị chuyến bay của Admin.

🛠 Công nghệ sử dụng

Backend: ASP.NET MVC 


Database: SQL Server 


Tools: Visual Studio, SQL Server Management Studio (SSMS), Excel (viết Test Case).

🌟 Các chức năng chính

Hệ thống: Đăng nhập, phân quyền (User/Admin).


Nghiệp vụ: Đặt vé máy bay, quản lý danh sách chuyến bay.

🔍 Quy trình Kiểm thử (Testing Process)

Đây là phần quan trọng nhất để chứng minh năng lực Tester của bạn.

1. Các loại hình kiểm thử đã thực hiện:


Manual Testing: Kiểm thử chức năng (Functional Testing) và kiểm thử giao diện (UI Testing).


Database Testing: Kiểm tra tính toàn vẹn và đúng đắn của dữ liệu trên SQL Server.

2. Tài liệu kiểm thử (Deliverables):

[Link bộ Test Case mẫu (Excel/Google Sheets)] https://docs.google.com/spreadsheets/d/19r6N9tLOGB_CHpYC40DCZ-jWPS5EXz5hgC7RjthXqcg/edit?usp=sharing

3. Phạm vi kiểm thử:

Kiểm tra logic phân quyền: Đảm bảo User không thể truy cập vào các trang quản trị của Admin.

Kiểm tra luồng nghiệp vụ end-to-end: Từ lúc khách chọn chuyến bay đến khi lưu thông tin vào DB.

Kiểm tra validate dữ liệu: Các trường thông tin trống, định dạng email, ngày tháng.
